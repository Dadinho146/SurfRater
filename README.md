# SurfRater

SurfRater is a .NET-based application designed to help surfers rate and review surf spots, track conditions, and share insights with the community.

## 🛠 Plan on Short Run

- Math Model Development and Test
- Integrate surf condition APIs
- Add a .NetMaui frontend (Android/Windows)

## 🛠 Plan on Long Run

- Add a .NetMaui frontend (Android/iOS/Windows)
- Release it on google playstore and apple store
- User authentication and profiles
- Make a ton of money from it and help ocean wild life from it
